# uploadImage

SSM上传文件项目Demo

#### 使用说明

1.  IDEA导入项目后修改maven、tomcat、jdk配置，然后maven自动下载包
2. 数据库配置文件自行修改
3. 在tomcat的配置文件中设置一个用来放置图片的虚拟目录，具体操作使用博客的server.xml配置方式，博客地址：https://blog.csdn.net/alela_/article/details/104466476
